Python generator in `generators/python`.  
JS generator in `generators/web` and online at [autre.space/sisifont](https://autre.space/sisifont)

─ │ ┌ ┐ └ ┘ ├ ┤ ┬ ┴ ┼ ╭ ╮ ╯ ╰ ╴ ╵ ╶ ╷ ~ ^ ○ · / \
